# 🛟 Social Hub - Dashboard de Campanhas

Dashboard inteligente para gestão de redes sociais com IA.

## ✨ Funcionalidades

- 📊 **Dashboard** - Visão geral com métricas consolidadas
- 📅 **Agenda** - Calendário visual para programações de posts
- 🚀 **Campanhas** - Gestão completa de campanhas de marketing
- 💡 **Insights** - Análises e sugestões geradas por IA

## 🛠️ Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Recharts (gráficos)
- Lucide React (ícones)
- date-fns (datas)

## 🚀 Como usar

```bash
# Instalar dependências
pnpm install

# Rodar em desenvolvimento
pnpm dev

# Build para produção
pnpm build
```

## 🔌 APIs e Integrações

O projeto está preparado para conectar com:
- Meta Graph API (Instagram, Facebook)
- TikTok API
- LinkedIn API
- Twitter/X API

## 📝 Estrutura

```
src/
├── components/    # Componentes reutilizáveis
├── pages/        # Páginas do dashboard
├── types/        # Tipos TypeScript
├── hooks/        # Custom hooks
├── services/     # Integração com APIs
└── data/         # Dados mockados
```
